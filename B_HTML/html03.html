II.2 HTML Encabezados, Párrafos, Saltos de Línea

II.2.1 HTML.Encabezados

Los encabezados son elementos HTML que se utilizan para definir titulares o cabeceras de un documento o página web. Existen seis niveles de encabezados y van desde el <h1> hasta el <h6>, ordenados de mayor a menor importancia.

Esto es un título de <h1>

<h1>Esto es un título</h1>

El elemento <h1> es el más importante del documento, ya que establece el título principal. Los encabezados secundarios (subtítulos) se introducen con los elementos <h2>, <h3>, <h4>, <h5> y <h6>.

En el siguiente ejemplo se muestran todos los niveles de encabezados:

Encabezado de primer nivel.

Encabezado de segundo nivel.

Encabezado de tercer nivel.

Encabezado de cuarto nivel.

Encabezado de quinto nivel.
Encabezado de sexto nivel.

Estructura correcta de encabezados

Debe existir un único elemento <h1> en la página ó documento. Más de uno puede crear confusión sobre el tema principal de la página. El resto de niveles pueden usarse múltiples veces.

Ejemplo correcto de estructura:


        <h1>Artículo sobre mascotas</h1>
         <h2>Los gatos</h2>
          <h3>El gato persa</h3>
          <h3>El gato siamés</h3>
        <br>
         <h2>Los perros</h2>
          <h3>El pastor alemán</h3>
          <h3>El caniche</h3>
        <br>
         <h2>Artículos relacionados</h2>

Artículo sobre mascotas

Los gatos

El gato persa

El gato siamés


Los perros

El pastor alemán

El caniche


Artículos relacionados

Errores comunes

No es correcto usar un <h4> por encima de un <h3>, ni un <h2> como nivel superior de un <h1>.

Ejemplo incorrecto


        <h3>Título del artículo</h3>
        <h1>Texto más grande</h1>

Título del artículo

Texto más grande

Uso correcto de estilos

Dado que el tamaño del texto cambia según el nivel de encabezado (<h1> es el más grande y <h6> el más pequeño), algunos diseñadores los usan solo por estética. Esto no debe hacerse. Para modificar el tamaño de fuente debe usarse CSS.

II.2.2 HTML.Párrafos

Los párrafos son elementos que ofrecen una manera de estructurar el texto en diferentes bloques de texto. En la literatura, los párrafos se inventaron con la finalidad de separar ideas visualmente. Por tanto, este elemento HTML imita la misma finalidad. El texto debe ir entre las etiquetas <p> (de apertura) y </p> (de cierre).


        <p>Esto es un párrafo de siete palabras.</p>

Ejemplo de un párrafo en HTML:

Esto es un párrafo de siete palabras.

Ejemplo de un párrafo situado en la estructura básica de un documento HTML:


        <html>

          <head>

            <title>Título del documento</title>

          </head>

          <body>

            <h1>Este texto es el encabezado</h1>

            <p>Esto es un párrafo de siete palabras.</p>

          </body>

        </html>

Características de los párrafos

Al igual que en el caso de los encabezados, los párrafos también son elementos de bloque. Esto significa que empiezan en una línea nueva y además ocupan todo el ancho posible, impidiendo que quepa otro elemento a su lado.

Normas de uso

  • Los elementos de bloque no pueden contener otros elementos de bloque. Esto significa que los párrafos no pueden contener otros párrafos, ni encabezados, ni listas, ni divisiones (<div>), ni formularios (<form>), ni tablas (<table>).
  • No se aconseja el uso de párrafos vacíos, es decir, con las etiquetas de apertura y de cierre sin contenido dentro de ellas. Si se quiere borrar el contenido de un párrafo, es preferible borrar también las etiquetas contenedoras en vez de dejar este fragmento de código: <p></p> en el documento HTML o página web.

Estilo por defecto de los párrafos

El estilo CSS por defecto de los párrafos añade un margen de 1em por encima y 1em por debajo. Gracias a este margen y al hecho de que las líneas de cada párrafo tienen menos margen, se consiguen separar visualmente unos de otros.

Atributos de párrafo

Los párrafos tienen los atributos esenciales que tienen la mayoría de elementos HTML, como id, class, title y style, además de los atributos de idioma (lang, xml:lang) y de los atributos de eventos para JavaScript. No tienen atributos especiales que ofrezcan características únicas para este tipo de elemento.

II.2.3 HTML.Saltos de línea

Los saltos de línea son elementos que permiten insertar una línea nueva (retorno de carro & nueva línea), por ejemplo, dentro de un mismo párrafo. Es un elemento muy útil cuando se necesita organizar la información en varias líneas, como sería el caso de un poema o una dirección postal. Se utiliza la etiqueta <br> y no tiene etiqueta de cierre, por lo que es un elemento clasificado como elemento vacío (void element).

Ejemplo de un salto de línea en HTML:

        <p>Recursivos: <br> Ciencia computacional</p>

El código de arriba se verá así:

Recursivos: Ciencia computacional

Normas de uso

A menudo se utiliza el salto de línea para incrementar el espacio entre líneas de texto, y en general es una mala práctica. La forma correcta de incrementar el espacio entre objetos es usando las hojas de estilo CSS, concretamente la propiedad margin, ya que permite modificar el margen visual entre elementos.

Oportunidad de salto de línea: Elemento

Con el elemento <wbr> se puede crear una oportunidad de salto de línea. Este elemento es de utilidad para crear un salto de línea de forma opcional, de modo que el navegador web hará el salto de línea solo en caso de que la palabra no quepa en el contenedor; la palabra se verá dividida.

En el siguiente ejemplo se escribe una URL ficticia con varios saltos de línea opcionales en sus distintos niveles de página.


        <p>https://example.com/<wbr>categoria/<wbr>pagina/<wbr>categoria/<wbr>pagina/<wbr>categoria/<wbr>pagina/<wbr>categoria/<wbr>pagina/<wbr>categoria/<wbr>pagina/</p>

Dando como resultado:


        
https://example.com/categoria/pagina/categoria/pagina/categoria/pagina/categoria/pagina/categoria/pagina/

https://example.com/categoria/pagina/categoria/pagina/categoria/pagina/categoria/pagina/categoria/pagina/

Tal y como se puede ver en el ejemplo anterior, el navegador ha hecho el salto de línea de forma opcional en uno de los puntos que se le ha indicado.
