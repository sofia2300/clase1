II.13 HTML.Ejercicios

1. Estructura básica

Cree un documento HTML básico con las etiquetas necesarias: doctype, html, head, title y body. Dentro del body agregue un encabezado <h1> y un párrafo.

2. Encabezados

Escriba una página que contenga todos los tipos de encabezados desde <h1> hasta <h6>. Use títulos relacionados con temas de interés general.

3. Párrafos

Escriba tres párrafos sobre un tema que le interese (por ejemplo: salud, historia o medio ambiente) y uno con lorem ipsum de 25 paralabras. Use la etiqueta <p> para cada uno.

4. Formateo de texto

Cree un texto donde use las siguientes etiquetas: <strong>, <em>, <del>, <ins>, <small> y <u>.

5. Listas no ordenadas

Cree una lista no ordenada con cinco elementos relacionados con un tema profesional o cultural, por ejemplo: herramientas manuales, libros importantes o alimentos saludables.

6. Listas ordenadas

Haga una lista ordenada con los pasos para preparar una receta sencilla o para cambiar una llanta del auto.

7. Imágenes

Inserte, dedes lorem picsum, una imagen utilizando la etiqueta <img>. Use un atributo alt y defina el ancho y alto de la imagen.

8. Enlaces

Agregue un enlace a una página web de interés público. Haz que se abra en otra pestaña.

9. Listas anidadas

Cree una lista que tenga categorías y sublistas. Por ejemplo: deportes y sus modalidades, o comidas y sus ingredientes principales.

10. Tablas simples

Diseñe una tabla con dos columnas y tres filas. Puede ser una tabla con información como “Nombre” y “Cargo”, o “Producto” y “Precio”.

11. Atributos en tablas

En la tabla del ejercicio anterior, agrega bordes, encabezados <th> y usa los atributos rowspan o colspan al menos una vez.

12. Formularios básicos

Crea un formulario simple con los campos: nombre, correo electrónico y mensaje. Usa las etiquetas <form>, <input> y <textarea>.

13. Tipos de campos

Agrega al formulario del ejercicio anterior campos de tipo contraseña, número, fecha y checkbox.

14. Botones de envío

Completa el formulario agregando un botón de envío con texto personalizado y otro botón de reinicio.

15. Etiquetas y agrupación

Usa la etiqueta <label> para asociar cada campo del formulario con su descripción. Agrupe contenido usando <fieldset> y <legend>.

16. Elementos multimedia

Inserta un audio o video usando las etiquetas <audio> o <video>. Asegúrate de incluir controles y una fuente válida.

17. Div y span

Usa las etiquetas <div> y <span> para organizar bloques de texto y aplicar estilos básicos de forma semántica. Los textos obténgalos de lorem ipsum

18. Comentarios en HTML

Agrega comentarios dentro del código HTML para explicar partes importantes del documento. Use la sintaxis .

19. Metadatos

En la sección <head> de una página, agrega metadatos como charset, description y keywords usando la etiqueta <meta>.

20. Integración final - Página sobre un animal

Cree una página web dedicada al dodo. Incluya las siguientes secciones:

  • Introducción: Breve descripción del dodo.
  • Hábitat: Lugar donde vivía, condiciones naturales.
  • Alimentación: Qué comía el dodo.
  • Extinción: Causas por las que se extinguió.
  • Galería: Inserte mínimo dos imágenes desde una carpeta local llamada fotos/.

Use todo lo aprendido: estructura básica, texto formateado, encabezados, párrafos, imágenes, enlaces, listas, tablas, formularios y elementos multimedia si es posible. Si necesita párrafos use lorem ipsum.
