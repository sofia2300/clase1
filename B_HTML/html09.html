II.8.1.1 HTML.Campo de texto simple (<input type="text">)

Permite al usuario ingresar una línea de texto.

Atributos:

Nombre Explicación Opciones
type Define el tipo del elemento input. text (por defecto), password, email, number, url, tel, etc.
name Nombre del control, usado para enviar los datos del formulario.
id Identificador único del elemento.
value Valor inicial del campo.
placeholder Texto de sugerencia que aparece cuando el campo está vacío.
required Especifica que el campo debe ser llenado antes de enviar el formulario.
maxlength Especifica la longitud máxima de caracteres permitidos.

Código Ejemplo:


            <label for="nombre">Nombre:</label>
            <input type="text" id="nombre" name="nombre" placeholder="Tu nombre aquí" required>

II.8.2 HTML.Área de texto (<textarea>)

Permite al usuario ingresar múltiples líneas de texto.

Atributos:

Nombre Explicación Opciones
name Nombre del control, usado para enviar los datos del formulario.
id Identificador único del elemento.
rows Especifica el número visible de líneas en un área de texto.
cols Especifica el ancho visible de un área de texto.
placeholder Texto de sugerencia que aparece cuando el área de texto está vacía.
required Especifica que el campo debe ser llenado antes de enviar el formulario.
maxlength Especifica la longitud máxima de caracteres permitidos.

Código Ejemplo:


            <label for="mensaje">Mensaje:</label>
            <textarea id="mensaje" name="mensaje" rows="4" cols="50" placeholder="Escribe tu mensaje aquí"></textarea>
Resultado:

II.8.3 HTML.Lista desplegable (<select>)

Permite al usuario seleccionar una opción de una lista.

Atributos del elemento <select>:
Nombre Explicación Opciones
name Nombre del control.
id Identificador único.
multiple Permite seleccionar múltiples opciones.
required Indica que se debe seleccionar una opción.

Atributos del elemento <option>:

Nombre Explicación Opciones
value Valor que se envía al servidor cuando se selecciona la opción.
selected Especifica que la opción debe estar preseleccionada al cargar la página.
Código Ejemplo:

            <label for="pais">País:</label>
            <select id="pais" name="pais">
            <option value="co">Colombia</option>
            <option value="mx">México</option>
            <option value="es" selected>España</option>
            <option value="ar">Argentina</option>
            </select>

Resultado:

II.8.4 HTML.Casilla de verificación (<input type="checkbox">)

Permite al usuario seleccionar una o varias opciones de un conjunto.

Atributos:

Nombre Explicación Opciones
type Define el tipo del elemento input. checkbox
name Nombre del grupo de casillas de verificación (para enviar múltiples valores).
id Identificador único.
value Valor que se envía al servidor si la casilla está marcada.
checked Especifica que la casilla debe estar marcada al cargar la página.
required Hace que al menos una casilla del grupo sea obligatoria (dependiendo del navegador).
Código Ejemplo:

            <label><input type="checkbox" name="intereses" value="musica"> Música</label><br>
            <label><input type="checkbox" name="intereses" value="deporte" checked> Deporte</label><br>
            <label><input type="checkbox" name="intereses" value="lectura"> Lectura</label>

Resultado:



II.8.5 HTML.Botón de radio (<input type="radio">)

Permite al usuario seleccionar una única opción de un conjunto.

Atributos:

Nombre Explicación Opciones
type Define el tipo del elemento input. radio
name Nombre del grupo de botones de radio (para asegurar que solo uno esté seleccionado).
id Identificador único.
value Valor que se envía al servidor cuando el botón está seleccionado.
checked Especifica que el botón debe estar seleccionado al cargar la página.
required Hace que al menos un botón del grupo sea obligatorio (dependiendo del navegador).

Código Ejemplo:


            <label><input type="radio" name="genero" value="masculino"> Masculino</label><br>
            <label><input type="radio" name="genero" value="femenino" checked> Femenino</label><br>
            <label><input type="radio" name="genero" value="otro"> Otro</label>

Resultado:

II.8.8 HTML.Control de rango (<input type="range">)

Permite al usuario seleccionar un valor dentro de un rango.

Atributos:

Nombre Explicación Opciones
type Define el tipo del elemento input. range
name Nombre del control.
id Identificador único.
min Especifica el valor mínimo permitido.

II.8.9 HTML.Control de archivo (<input type="file">)

Permite al usuario seleccionar uno o varios archivos.

Atributos:

Nombre Explicación Opciones

II.8.11 HTML.Sugerencias de entrada (<datalist>)

Proporciona una lista de opciones predefinidas para un elemento <input>.

Atributos del elemento <datalist>:

Nombre Explicación Opciones
id Identificador único del datalist, referenciado por el atributo list del input.

Atributos del elemento <option> dentro de <datalist>:

Nombre Explicación Opciones
value Valor de la opción.
Código Ejemplo:

            <label for="navegador">Elige un navegador:</label>
            <input list="browsers" id="navegador" name="navegador">
            <datalist id="browsers">
            <option value="Internet Explorer">
            <option value="Firefox">
            <option value="Chrome">
            <option value="Opera">
            <option value="Safari">
            </datalist>

Resultado:

II.8.12 HTML.Salida de resultados (<output>)

Muestra el resultado de un cálculo u otra salida relacionada con el formulario.

Atributos:

Nombre Explicación Opciones
name Nombre del control.
for Lista de IDs de otros elementos a los que está relacionado el resultado.

Código Ejemplo:


            <form oninput="resultado.value=parseInt(a.value)+parseInt(b.value)">
            <input type="number" id="a" name="a" value="0"> +
            <input type="number" id="b" name="b" value="0"> =
            <output name="resultado" for="a b">0</output>
            </form>

Resultado:

+ = 0

II.8.13. HTML.Elementos de Formulario: Label, Fieldset, Legend

II.8.13.1. HTML.Etiqueta (<label>)

Define una etiqueta para un elemento de formulario, mejorando la accesibilidad y usabilidad.

Atributos:

Nombre Explicación Opciones
for Especifica el id del elemento de formulario al que está asociada la etiqueta. Un valor de id existente en el documento.

Código Ejemplo: 

            <label for="nombre">Nombre:</label>
            <input type="text" id="nombre" name="nombre">

Resultado:

II.8.13.2 HTML.Grupo de campos (<fieldset>)

Agrupa elementos relacionados dentro de un formulario.

Atributos:

Nombre Explicación Opciones
disabled Especifica que el grupo de campos debe estar deshabilitado.
form Especifica a qué formulario pertenece el grupo de campos (si no está anidado directamente dentro de un formulario). El id de un elemento <form>.
name Nombre del grupo de campos.
Código Ejemplo:

            <fieldset>
            <legend>Información Personal</legend>
            <label for="nombre_fs">Nombre:</label><br>
            <input type="text" id="nombre_fs" name="nombre_fs"><br><br>
            <label for="email_fs">Email:</label><br>
            <input type="email" id="email_fs" name="email_fs"><br><br>
            </fieldset>

Resultado:

Información Personal





II.8.13.3 HTML.Leyenda de grupo de campos (<legend>)

Define un título para el contenido del elemento <fieldset>.

Atributos:

Nombre Explicación Opciones
No tiene atributos específicos relevantes para su función principal.
