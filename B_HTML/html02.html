Explicación Documento 01

La primera línea <!DOCTYPE html> es la declaración de la versión del tipo de documento: HTML.

Indica al navegador web que este documento es HTML5. No distingue entre mayúsculas y minúsculas.

---

El elemento <html> es el elemento raíz de un documento HTML.

El atributo lang="en" indica que el contenido del archivo va a estar en inglés.

---

El elemento <head> es un contenedor para las etiquetas que proporcionan información sobre el documento, por ejemplo:

<title> define el título del documento en la pestaña del navegador.

<meta charset="UTF-8"> indica que el documento utiliza la codificación de caracteres UTF-8, permitiendo mostrar correctamente caracteres especiales y acentuados.

<meta name="viewport" content="width=device-width, initial-scale=1.0"> permite que la página sea responsiva, adaptando su tamaño y escala a diferentes dispositivos, especialmente móviles.

---

El elemento <body> contiene el contenido real del documento (párrafos, enlaces, imágenes, tablas, etc.) que se representa en el navegador web y se muestra al usuario.

El elemento <p> es un ejemplo de los diferentes elementos en HTML, que permiten desplegar información al usuario, en este caso, permite escribir un párrafo dentro del body

---

> Nota: La declaración DOCTYPE aparece en la parte superior de una página web, antes que cualquier otro elemento; sin embargo, la declaración DOCTYPE en sí no es una etiqueta HTML. Todo documento HTML requiere una declaración de tipo de documento para garantizar que las páginas se muestren correctamente. Los tags etiquetas <html>, <head>, y <body> conforman la estructura básica de cada página web. El contenido dentro de las etiquetas <head>y </head> es invisible para los usuarios, con una excepción: el texto entre <title> y </title> que aparece como título en la pestaña correspondiente del navegador.

II.1.2. HTML.TAGS & Elementos

HTML se escribe en forma de elementos HTML que consisten en etiquetas (tags) de marcado.

Estas etiquetas son la característica fundamental de HTML. Cada etiqueta de marcado se compone de una palabra clave,

entre corchetes angulares, signos de menor & mayor (< y >), como por ejemplo: <html>, <head>, <body>, <title>, <p>, etc.

Estructura de un elemento HTML:

---

Las etiquetas HTML suelen venir en pares, como <html>y </html>. La primera etiqueta de cada par suele llamarse

etiqueta de apertura (o etiqueta inicial), y la segunda, etiqueta de cierre (o etiqueta final).

Una etiqueta de apertura y una etiqueta de cierre son idénticas, excepto por una barra ( /) después del corchete angular de

apertura de la etiqueta de cierre, para indicarle al navegador que el comando se ha completado.

Entre las etiquetas de inicio y fin se puede colocar el contenido apropiado. Por ejemplo, un párrafo, se escribiría así:


                <p>Este es un párrafo.</p>
                <!-- Este es un comentario para un párrafo que tiene un elemento anidado -->
                <p>
                    Este es <b>otro</b> párrafo con un elemento de negrilla.
                </p>

II.1.6 HTML.Tipos de Elementos HTML

Los elementos se dividen en dos grupos: de tipo bloque y de tipo inline.

  • Los elementos de tipo bloque ocupan el 100% del ancho disponible. Ejemplos: <div>, <p>, <h1>.
  • Los elementos de tipo inline ocupan solo el espacio necesario. Ejemplos: <img>, <a>, <span>.

Los elementos de tipo bloque pueden contener otros elementos de tipo bloque o inline, mientras que los elementos de tipo inline solo pueden contener otros elementos de tipo inline.

Ejemplo de elemento de tipo bloque:


            <div>
                <p>Este es un párrafo dentro de un div.</p>
            </div>

Este es un párrafo dentro de un div.

Ejemplo de elemento de tipo inline:


            <span>Este es un texto dentro de un span.</span>
Este es un texto dentro de un span.

II.1.7 HTML.Atributos

Los atributos HTML proporcionan información adicional sobre un elemento HTML. Se colocan dentro de la etiqueta de apertura.

Los atributos HTML tienen un nombre y un valor, separados por un signo igual (=).

Ejemplo de un atributo HTML:


            <a href="https://picsum.photos/">Visitar ejemplo</a>
            <img src="https://picsum.photos/200/300" alt="Imagen aleatoria de Lorem Picsum">
            <input type="text" name="username">       
            <input type="password" name="password">
            <input type="submit" value="Enviar">
            <input type="reset" value="Restablecer">
            <input type="checkbox" name="opcion1"> Opción 1

Los atributos type, name, value, src, y alt son ejemplos de atributos HTML.

El atributo href se utiliza para especificar la URL de un enlace.

El atributo src se utiliza para especificar la URL de una imagen.

El atributo alt proporciona un texto alternativo para la imagen, que se muestra si la imagen no se puede cargar.

El atributo type se utiliza para especificar el tipo de entrada en un formulario.

El atributo name se utiliza para identificar un elemento en un formulario.

El atributo value se utiliza para especificar el valor de un elemento de formulario.

El atributo checked se utiliza para especificar que un elemento de formulario debe estar seleccionado por defecto.

Los atributos HTML son sensibles a mayúsculas y minúsculas, por lo que src y Src son diferentes.

Los atributos HTML pueden tener valores vacíos, como en el caso de los atributos booleanos.

Los atributos HTML pueden tener valores de cadena, como en el caso de los atributos href, src, y alt.

Los atributos HTML pueden tener valores numéricos, como en el caso de los atributos width y height.

II.1.7.1 HTML.Atributos Booleanos

Algunos atributos no consisten en pares nombre/valor sino solo en un nombre. Estos se llaman atributos booleanos.


            <input type="email" required>
            <input type="submit" value="Submit" disabled>
            <input type="checkbox" checked>
            <input type="text" value="Read only text" readonly>

II.1.7.2 HTML.Elementos Referencias

Los elementos HTML son las partes de contenido que componen un documento.A través de los elementos se inserta texto, imágenes o vídeo en una página web.

Elementos de bloque

Los elementos se pueden clasificar en dos grupos, según el espacio que ocupan en la página. Algunos elementos son de bloque y otros son elementos en línea. Los elementos de bloque siempre empiezan en una línea nueva y ocupan todo el espacio disponible hasta el final de la línea. Esto significa que ocupan la anchura máxima hasta encontrar los límites del elemento contenedor superior. Estos elementos no permiten que otro elemento se pueda colocar a su lado.

Lista de los elementos de bloque

<address>, <blockquote>, <center>, <dd>, <div>, <dl>, <dt>, <fieldset>, <form>, <h1> - <h6>. <hr>, <li> <noscript> <ol> <p> <pre> <table> <tbody> <tfoot> <tr> y <ul>

El elemento <div>

El elemento <div> se utiliza para crear secciones o agrupar contenido; se trata de un elemento contenedor muy importante a nivel de diseño web. Este elemento no proporciona un cambio visual en el bloque, pero tiene su utilidad cuando se usa con las hojas de estilo CSS. Por tanto, todos los elementos que se quieran agrupar deberán ir dentro de las etiquetas <div> (de apertura) y </div> (de cierre).

Ejemplo de código HTML usando un elemento <div>

            
                <div>
                    <p>Los lenguajes están en orden alfabético.</p>
                    <ol>
                        <li>Bash</li>
                        <li>C</li>
                        <li>Go</li>
                        <li>Perl</li>
                        <li>Python</li>
                        <li>Ruby</li>
                    </ol>
                </div>

Elementos de línea

Los elementos de linea ocupan el mínimo espacio horizontal, y de esta manera permiten que otros elementos de línea se coloquen a su lado. Esto significa que este tipo de elementos ocupan únicamente el espacio necesario para mostrar su contenido. El ejemplo más significativo son los enlaces.

Lista de los elementos de linea

<a> <abbr> <acronym> <b> <bdo> <big>, <br>, <button>, <cite> <code> <del>, <dfn> <em> <i> <iframe>, <img>, <input>, <ins> <kbd>, <label>, <map>, <mark>, <object> <q>, <samp>, <script>, <select>, <small> <span> <strong> <sub> <sup>, <textarea>, <tt> y <var>

El elemento <span>

El elemento <span> se utiliza para agrupar elementos en línea o simplemente aplicar estilo al texto combinándolo con los atributos class e id. Este elemento tampoco proporciona ningún cambio visual, pues tiene su utilidad cuando se usa con las hojas de estilo CSS, Por tanto, todos los elementos de linea que se quieran agrupar deberán ir entre las etiquetas <span> (de apertura) y </span> (de cierre).

Ejemplo de código HTML usando <span>

<p>Este párrafo tiene texto <span style='color:green'>verde</span>.</p>

Dando como resultado:

Este párrafo tiene texto verde.

En este ejemplo se ha usado el atributo style para indicar el color verde mediante el uso de hojas de estilo en cascada (CSS). En el valor de style se introducen las declaraciones CSS, compuestas por una propiedad y un valor.

Uso de elementos según tipología

Saber diferenciar los dos tipos de elementos es imprescindible para poder hacer un buen diseño y estructurar el contenido de forma correcta. Además, la mayoría de elementos de bloque pueden contener otros elementos de bloque y elementos de línea. Pero los elementos de línea sólo pueden contener otros elementos de línea.

Resumiendo, la principal diferencia entre el elemento <div> y el elemento <span> es que el primero se usa para agrupar elementos en un bloque y el segundo se usa para agrupar elementos en una línea. Es lógico, pues, que este último sólo pueda contener elementos de línea.
